********************** The Programs **********************
**********************************************************

The programs available are R programs. They were written in 
version 2.6.1 of R. 

1) "estimateN.r"
This file contains various functions used in model
fitting. It is accessed by other programs.
(ASCII text)

2) "example1.r"
An example of how to fit various models
(ASCII text)

3) "example2.r"
Further examples of how to fit various models
(ASCII text)

4) "estimateN.pdf"
Detailed instructions for using the software and 
some warnings about possible limitations.
(PDF format)
