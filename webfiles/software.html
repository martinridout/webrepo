Software

The links on this page provide access to software related to some of my recent research interests. Most of the code is in R, but there is also some Matlab and Maple. Data sets are also available in several instances.

Statistical ecology

  • R code for fitting mixture models to capture-recapture data
  • R and Maple code relating to perturbation analysis of matrix population models
  • R package for estimating overlap of activity patterns
  • SODA a software tool to assist in the design of single-season occupancy studies

Other statistical things

  • R functions for the tempered stable distribution
  • R code for simulating random numbers from a continuous distribution specified by its Laplace transform
  • R code for complex-step differentiation

Bioscience

  • The separate prion pages have software for calculating prion curing curves (no link back to these pages).
