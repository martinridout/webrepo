Genetic Linkage Analysis

My interests in linkage analysis have been in connection with outcrossing plant species. I have overseen the development of three software packages, which are freely available:

  • LINKEM - a program for doing two-point linkage analysis for data from a single cross
  • LINK3EM - an extension of LINKEM that does three-point linkage analysis.
  • POLYLINK - a program to assist in linkage analysis for autopolyploids.
