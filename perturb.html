Programs for perturbation analysis of matrix population models

These programs are related to a paper on exact perturbation analysis using symbolic algebra [Ref 1].

  • killerwhale.r - R code for the killer whale example (generates Fig 1 in paper).
  • lizardorchid1.r - R code for the lizard orchid example (generates Fig 2 in paper).
  • lizardorchid2.r - further R code for the lizard orchid example (generates Fig 3 in paper).
  • killerwhale.mws - the Maple code (Maple 11) for the killer whale example.

Reference

[1] Miller, D., Morgan, B.J.T., Ridout, M.S., Carey, P.D. and Rothery, P. (2011) Methods for explicit perturbation analysis. Methods in Ecology and Evolution, 2, 283-288. [Journal link]
